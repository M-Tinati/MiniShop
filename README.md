# 🛒 MiniShop



در طول توسعه این پروژه، مفاهیمی مانند **برنامه‌نویسی شی‌گرا (OOP)**، **طراحی پایگاه داده**، **Clean Code**، **معماری نرم‌افزار** و کار با **WindowsForm** و **سی شارپ** به‌صورت عملی پیاده‌سازی می‌شود.

این پروژه به‌صورت مرحله‌به‌مرحله توسعه پیدا می‌کند و با اضافه شدن هر قابلیت، ساختار پروژه نیز استانداردتر و قابل توسعه‌تر میشود.
